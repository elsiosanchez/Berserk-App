## 🔴 ABOUT

**BERSERK is a 4-week hypertrophy program designed for serious lifters who want structured progression, real variety, and zero excuses.** Every week brings new exercises for the same muscle groups so your body never adapts and your mind never gets bored.

Built on the Push/Pull/Legs method with 6 training days per week. Each session includes guided warm-ups, gym exercises with rest timers, YouTube tutorials, and a complete home/park alternative for days you can't make it to the gym.
